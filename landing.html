How It Works
From zero to context in 4 steps
No infrastructure to set up. No models to fine-tune. Just API calls.
1

Get your API key

Sign up with your email. Instant key, no credit card. Start on the free tier with 1,000 queries/month.

2

Create a project

A project is a knowledge namespace. One per product, team, or use case. Keeps context isolated and relevant.

3

Add sources

Point us at URLs, upload files, or paste text. We auto-crawl, chunk, and embed everything. JS-rendered sites included.

4

Query from your agent

One POST request returns ranked, relevant context ready to inject into any LLM's system prompt. That's it.

Features
Everything your agent needs to be smart
Built for production AI agents. Not a toy demo.
⚡

Hybrid Search

Dense vector + BM25 sparse retrieval. Better recall than vector-only search, especially for technical content.

🌐

Auto-Crawling

Point us at a URL or sitemap. We crawl, parse, and re-index on schedule. Handles JS-rendered sites with Playwright.

📈

Smart Chunking

Context-aware chunking preserves document structure. Section headings, code blocks, and lists stay intact.

💡

Freshness Re-ranking

Recent content ranks higher automatically. Stale docs don't pollute your agent's answers.

🔎

Source Discovery

Tell us a topic — we find relevant URLs across the web and auto-ingest them into your project.

📊

Query Analytics

See what your users ask, which chunks get returned, latency percentiles. Understand and improve your agent's knowledge.

🔒

Project Isolation

Each project has its own vector space. No cross-contamination between products, tenants, or environments.

🚀

One API Call

Single POST endpoint returns ranked chunks + a ready-to-use context string. No client-side ranking or assembly.

🛠

Dashboard

Web UI to manage projects, sources, and monitor queries. No code needed for content operations.
