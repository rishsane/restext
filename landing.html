Features
Everything your agent needs to be smart
Built for production AI agents. Not a toy demo.
⚡

Hybrid Search

Dense vector + BM25 sparse retrieval. Better recall than vector-only, especially for technical content and exact terms.

🌐

Auto-Crawling

Point us at a URL or sitemap. We crawl, parse, and re-index on schedule. JS-rendered sites handled with Playwright.

📈

Smart Chunking

Context-aware chunking preserves document structure. Section headings, code blocks, and lists stay intact.

💡

Freshness Re-ranking

Recent content ranks higher. Stale docs don't pollute answers. Time-weighted scoring built in.

🔎

Source Discovery

Tell us a topic — we find relevant URLs across the web and auto-ingest them into your project.

📊

Query Analytics

See what users ask, which chunks return, latency percentiles. Understand and improve your agent's knowledge.

🔒

Project Isolation

Each project has its own vector space. No cross-contamination between products, tenants, or environments.

🚀

One API Call

Single POST returns ranked chunks + a ready-to-use context string. No client-side ranking or assembly needed.

🛠

Web Dashboard

Manage projects, sources, and monitor queries from a clean web UI. No code needed for content operations.
